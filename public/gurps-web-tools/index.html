GURPS Web Tools

This web app is a collection of tools to help make your preperation and gameplay in GURPS pen and paper role playing game run much more smoothly.

Flexible Dice Roller

Roll some Bones. This little tool should be able to handle pretty much any natural combination you throw at it.

GM Control Sheet

This is an electronic GM Control Sheet. It allows for saving, loading and exporting of different encounters.

Character Generator

Make your hero.
